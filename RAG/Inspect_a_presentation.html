Inspect a presentation

Before you share a presentation, especially with people outside your organization, remove any personal and confidential information stored in the document or in the document properties.

  1. Open the presentation file that you want to inspect for hidden data and personal information.

    Tip: It’s a good idea to inspect a copy of your original document, because it’s not always possible to restore the data that the Document Inspector removes.

  2. Select File > Info .

  3. Select Check for Issues > Inspect Document .

  4. Select the types of content that you want to inspect, and then select Inspect .

  5. Review the results.

  6. For each type of content that you want to remove from the document. Often Comments or Document Properties and Personal Information are removed. Select the Remove All button from each section you want removed.

  7. Select Reinspect > Inspect to make sure all you want was removed.

  8. Select Close and then share your presentation.
